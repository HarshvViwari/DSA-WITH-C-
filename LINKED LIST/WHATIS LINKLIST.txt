               LINK LIST 
A LINK LIST IS A SPECIAL TYPE OF DATA STRACTURE 
WHICH CONTAIN A NODE WERE TWO TYPES OF DATA IS 
PRESENT

-> THE FIRST ONE IS DATA OF ANY TYPE
-> SECOND IT CONTAIN THE ADRESS OF THE ANOTHER NODE 
   OF THE LINKLIST 

TYPES OF LINK LIST
1. LINEAR LINK LIST
2. SINGLY CIRCULAR LINK LIST
3. DOUBLY LINK LIST 
4. DOUBLY CIRCULAR LINK LIST
